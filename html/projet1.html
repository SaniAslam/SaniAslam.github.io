Calculatrice

Projet 1

Voici un exemple de projet réalisé lors du premier semestre du BTS SIO.

Description

Ce projet est une calculatrice en Python réalisée dans le cadre du BTS SIO. Elle permet d’effectuer plusieurs opérations mathématiques de base : addition, soustraction, multiplication, division, puissance et modulo.

En plus des calculs classiques, la calculatrice propose un mode pour déterminer si un nombre est pair ou impair. L’utilisateur peut choisir le mode souhaité via un menu interactif, ce qui rend l’application simple et intuitive.

Code Python


def addition(a:float,b:float)->float:
    print("Le résultat de",a,"+",b,"est :",a + b )
def soustraction(a:float,b:float)->float:
    print("Le résultat de",a,"-",b,"est :",a - b )
def multiplication(a:float,b:float)->float:
    print( "Le résultat de",a,"*",b,"est :",a * b )
def division(a:float,b:float)->float:
    if b > 0.0:
        print( "Le résultat de",a,"/",b,"est :",a / b)
    else:
        print( "Division par 0 impossible")
def puissance(a:float,b:float)->float:
    print( "Le résultat de",a,"**",b,"est :",a ** b )
def modulo(a:float,b:float)->float:
     print( "Le résultat de",a,"%",b,"est :",a % b)

#Nombre pair ou impair 
def est_pair(n:int)->bool:
    if n % 2 == 0:
        return True
    else:
        return False

def afficher_parite():
    nombre = int(input("Entrez un nombre entier: "))
    parite = est_pair(nombre)
    if parite == True:
        print("Le nombre", nombre, "est pair")
    else:
        print("Le nombre", nombre, "est impair")

def racine_caree(a:float)->float:
    if a >= 0 :
        return a ** 0.5
    else:
        return "Racine carée avec un nombre négatif impossible"
    

def calculer(calcul:str)->float:
    elements = calcul.split()
    a = float(elements[0])
    operation = elements[1]
    b = float(elements[2])
    if operation == "+":
        return addition(a,b)
    elif operation == "-":
        return soustraction(a,b)
    elif operation == "*":
        return multiplication(a,b)
    elif operation == "/":
        return division(a,b)
    elif operation == "^":
        return puissance(a,b)
    elif operation == "%":
        return modulo(a,b)

def demander_calcul():
    calcul = input("Entrez un calcul(+, -, *, /, ^, %): ")
    calculer(calcul)
    
print("--- Calculatrice ---")
print("Quelle mode voulez vous utilisé ? :")
while True:
    mode = int(input("[1]->Calcul  [2]->Parité  [3]->Quitter  "))
    if mode == 1:
        demander_calcul()
    elif mode == 2:
        afficher_parite()
    elif mode == 3:
        print("Fin du programme")
        quit()
    else:
        print("Mauvaise saisie")
